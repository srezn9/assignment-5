Question-1: What are the different ways to select an element in the DOM?

There are multiple choices to select an element in DOM. There are getElementByID, getElementsByClass,getElementsByTagName, querySelector, querySelectorAll and so on.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns texts including html tags,while innerText returns texts , only visible on the screen ignoring html tags, and textContent returns all texts, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a method where a single event listener is added to a parent element instead of multiple listeners to each child. That makes the things way easier.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts at the target element and bubbles up to parents.

Question-5: How do you create, add, and remove elements using JavaScript?

document.createElement(), appendChild() to create and add elements and removeChild() to remove elements from DOM.
